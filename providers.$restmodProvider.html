<static> pushModelBase()

Adds mixins to the base model chain.

Non abstract models should NOT be added to this chain.

Base model chain is by default empty, all mixins added to the chain are prepended to every generated model.

Usage:

`javascript $provider.pushModelBase('ChangeModel', 'LazyRelations', 'ThrottledModel')
