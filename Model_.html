class:Model

new class:Model()

The base model type, this is the starting point for every model generated by $restmod. Inherits all static methods from Collection. ### Object creation Direct construction of object instances using `new` is not recommended. A collection of static methods are available to generate new instances of an model, for more information go check the Collection documentation.
