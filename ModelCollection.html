Class: ModelCollection

ModelCollection

new ModelCollection()

Collections are sets of model instances bound to a given api resource. When a model class is generated, the corresponding collection class is also generated.

Methods

$add(_obj) → {Collection}

Adds an item to the back of the collection. This method does not attempt to send changes to the server. To create a new item and add it use $create or $build.

Triggers after-add callbacks.
